## 🇩🇪 Deutsch

☢️ PS2 Vault-Tec BIOS Dump Utility ☢️
"Preparing for the Future, One Console at a Time."
Das Vault-Tec BIOS Dump Utility ist ein hochpräziser All-in-One-Extraktor für die PlayStation 2. Inspiriert von der legendären RobCo Industries Terminal-Technologie, bietet dieses Tool eine 100%ige Sicherung aller kritischen BIOS-Komponenten Ihrer Hardware.
Entwickelt von Holzer2009 mit Unterstützung modernster Google AI-Algorithmen.
🛠 Features

    Vollständiger 6-Teil-Dump: Extrahiert nicht nur den Standard-Kernel, sondern alle relevanten Schichten:
        ROM0 (Main BIOS - 4MB)
        ROM1 (Extended Data - 1MB)
        ROM2 (Sub-Data - 512KB)
        EROM (DVD Player / OS Updates - variabel)
        NVM (NVRAM / Systemeinstellungen - 1KB)
        MEC (MechaCon Controller Data)
    RobCo Termlink UI: Authentisches Fallout-Terminal-Interface mit CRT-Scanline-Effekten und taktiler Textausgabe.
    Hardware-Autodetect: Automatische Erkennung von Modellnummer (SCPH), Region (PAL/USA/JAP) und Konsolen-Revision (Fat/Slim).
    Safe-Guard System: Integrierte Prüfung auf USB-Speichermedien vor dem Schreibvorgang, um Datenverlust zu verhindern.

🚀 Installation & Nutzung

    Kopiere die VaultTec_Dump.elf auf einen FAT32-formatierten USB-Stick.
    Starte die Datei auf deiner PS2 über uLaunchELF (FreeMCBoot/MechaPwn/Modchip erforderlich).
    Folge den Anweisungen auf dem RobCo-Terminal.
    Drücke [X], um die Extraktionssequenz zu starten.
    Sobald die Meldung EXTRACTION COMPLETE erscheint, findest du deine Dumps sauber benannt auf deinem USB-Stick.

Durch die KI-gestützte Logik werden Adresskonflikte vermieden und ein valider Dump für Emulatoren wie PCSX2 garantiert.

## 🇺🇸 English Version

☢️ PS2 Vault-Tec BIOS Dump Utility ☢️
"Preparing for the Future, One Console at a Time."
The Vault-Tec BIOS Dump Utility is a high-precision all-in-one extractor for the PlayStation 2. Inspired by the legendary RobCo Industries terminal technology, this tool provides a 100% backup of all critical BIOS components from your hardware.
Developed by Holzer2009 with the support of cutting-edge Google AI algorithms.
🛠 Features

    Complete 6-Part Dump: Extracts not just the standard kernel, but all relevant layers:
        ROM0 (Main BIOS - 4MB)
        ROM1 (Extended Data - 1MB)
        ROM2 (Sub-Data - 512KB)
        EROM (DVD Player / OS Updates - variable)
        NVM (NVRAM / System Settings - 1KB)
        MEC (MechaCon Controller Data)
    RobCo Termlink UI: Authentic Fallout-style terminal interface with CRT scanline effects and tactile text output.
    Hardware Autodetect: Automatic detection of Model Number (SCPH), Region (PAL/USA/JAP), and Console Revision (Fat/Slim).
    Safe-Guard System: Integrated USB storage check before the writing process to prevent data loss.

🚀 Installation & Usage

    Copy the VaultTec_Dump.elf to a FAT32 formatted USB stick.
    Launch the file on your PS2 via uLaunchELF (FreeMCBoot/MechaPwn/Modchip required).
    Follow the instructions on the RobCo Terminal.
    Press [X] to initiate the extraction sequence.
    Once the message EXTRACTION COMPLETE appears, you will find your dumps properly named on your USB stick.
