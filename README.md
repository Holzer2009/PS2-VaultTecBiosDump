[Scroll down for English version](#english-version)

## 🇩🇪 Deutsch

Ein hochpräziser All-in-One Dumper für die PlayStation 2 mit Fallout-inspiriertem RobCo-Terminal. 
Entwickelt von **Holzer2009** (mit Unterstützung von Google AI).

### 🚀 Features
- **100% Extraction:** Sichert ROM0 (4MB), ROM1 (1MB), ROM2 (512KB).
- **Zusatzdaten:** Extrahiert NVM-Einstellungen, Mechacon-Daten (MEC) und EROM.
- **Auto-Erkennung:** Identifiziert automatisch SCPH-Modellnummer, Region (PAL/USA/JAP/ASIA) und V-Revision via ROMVER.
- **Interface:** Custom 8x8 Pixel-Font Rendering über `gsKit`.
- **USB-Support:** Direkter Export auf `mass:/` mit integrierter `usbhdfsd.irx`.

### ⚠️ Kompatibilität
Unterstützt alle **Retail-Modelle (V0 bis V19+)**. 
*Hinweis: Nicht geeignet für Net Yaroze oder DTL-Test-Hardware.*

## 🇺🇸 English Version

A high-precision all-in-one BIOS dumper for PlayStation 2 featuring a Fallout-inspired RobCo terminal interface. 
Developed by **Holzer2009** (with Google AI support).

### 🚀 Features
- **100% Extraction:** Dumps ROM0 (4MB), ROM1 (1MB), and ROM2 (512KB).
- **Additional Data:** Extracts NVM settings, Mechacon data (MEC), and EROM.
- **Auto-Detection:** Automatically identifies SCPH model number, region (PAL/USA/JAP/ASIA), and V-revision via ROMVER.
- **Interface:** Custom 8x8 pixel font rendering using `gsKit`.
- **USB Support:** Direct export to `mass:/` via integrated `usbhdfsd.irx`.

### ⚠️ Compatibility
Supports all **Retail models (V0 to V19+)**. 
*Note: Not compatible with Net Yaroze or DTL-Test hardware.*
